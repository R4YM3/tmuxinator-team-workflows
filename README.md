# Loop (oo)

Run any project-human or AI-generated-with one command.

Works with Docker, devcontainers, and your existing stack.
From AI-generated chaos to a predictable development workflow.

Loop (Launch Once, Operate Predictably) turns any project into a runnable, reliable dev environment in seconds.

No docs. No drift. No "works on my machine".

## ⚡ Start

```bash
oo
```

That's it.

Loop automatically:

- detects your project
- creates a workflow (if needed)
- installs missing requirements
- starts your services

No decisions. No flags. No friction.

## 🔄 Daily usage

```bash
oo
```

That's your default.

## 🛠 Commands (optional control)

```bash
oo add        # create workflow only
oo install    # install requirements only
oo start      # start workflow only
oo stop       # stop everything
oo doctor     # diagnose your environment
```

## 🧠 Smart command behavior

Running `oo` without arguments:

- no workflow -> creates one
- not installed -> installs requirements
- ready -> starts workflow

All in one flow. No interruptions.

## 🩺 Diagnose instantly

```bash
oo doctor
```

Find:

- missing dependencies
- incomplete setup
- misconfigured services

With clear next steps.
